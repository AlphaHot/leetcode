LeetCode - Customers Who Never Order

-----------+
| Id | Name  |
-----------+
| 1  | Joe   |
| 2  | Henry |
| 3  | Sam   |
| 4  | Max   |
-----------+

Table: Orders.

----------------+
| Id | CustomerId |
----------------+
| 1  | 3          |
| 2  | 1          |
----------------+

Using the above tables as example, return the following:

| Customers |

| Henry     |
